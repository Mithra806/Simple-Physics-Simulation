# Planetary Physics Sim

A Pygame simulation where balls bounce around, collide with each other, and can be dragged and thrown with your mouse.

![demo](demo.gif)
<!-- swap in your own gif/video -->

## Features
- Gravity + bouncing off walls
- Realistic elastic collisions between balls
- Click, drag, and throw any ball
- Each ball's weight changes depending on which planet it's on, since weight depends on both mass and that planet's gravity
- Live weight display, starfield background

## Run it
```bash
pip install pygame-ce
python main.py
```

## Controls
Click and hold a ball to grab it, drag to move it, release to throw it.
