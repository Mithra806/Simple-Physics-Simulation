# Planetary Physics Sim

A Pygame simulation where the 8 planets bounce around, collide with each other, and can be dragged and thrown with your mouse.

![demo](demo.gif)
<!-- swap in your own gif/video -->

## Features
- Gravity + bouncing off walls
- Realistic elastic collisions between planets
- Click, drag, and throw any planet
- Each planet has its own bounce/friction based on its real characteristics
- Live mass display, starfield background

## Run it
```bash
pip install pygame-ce
python main.py
```

## Controls
Click and hold a planet to grab it, drag to move it, release to throw it.
